## To run the web application, execute the command below

- Clone the project:
  ```bash
   git clone git@github.com:MateusAnderle/bookstore-full-stack-project.git
  ```

- Change to application folder:
  ```bash
   cd used-bookstore-nextjs 
  ```

- Install the dependencies:
  ```bash
  npm install
  ```

- Run application:
  ```bash
  npm run dev
  ```
