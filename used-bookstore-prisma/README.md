## To run the API, execute the command below

First, install de dependencies:

```bash
npm install
```

Then run :

```bash
npx prisma migrate dev --name Create DB
```

Start docker and execute this command:

```bash
sudo docker compose up
```

And finally, run the API:

```bash
npm start
```
